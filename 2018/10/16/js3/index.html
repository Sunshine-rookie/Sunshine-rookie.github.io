第 5 章

  • 5.27 操作方法

    1. splice（）方法：

      arrayObject.splice(index,howmany,item1,.....,itemX);
参数             描述
index         必需。整数，规定添加/删除项目的位置，使用负数可从数组结尾处规定位置。(起始位置)
howmany         必需。要删除的项目数量。如果设置为 0，则不会删除项目。（要删除的项数，0 表示不删除）
item1, ..., itemX    可选。向数组添加的新项目。（要插入的项，可以是多个）

splice()主要用途是向数组的中部插入项，有3种使用方法：
1）删除：可以删除任意数量的项，只需指定2个参数：要删除的第一项的位置和要删除的项数。如:splice(0,2)会删除数组中的前两项。
2）插入：可以向指定的位置插入任意数量的项，3个参数：起始位置、0（要删除的项数）和要插入的项。
   3）替换：可以向指定位置插入任意数量的项，且同事删除任意数量的项，只需指定3个参数：起始位置、要删除的项数和要插入的任意数量的项。插入的项数不必与删除的项数相等。

  • 5.63 String类型

    1.字符方法：

    两个用于访问字符串特定字符的方法：chart()和charCodeAt().这两个方法都接收一个参数，即基于0的字符位置。

var Str = 'hello world';

    2.字符串操作方法：

    concat(); 拼接字符串

    3.字符串位置方法：

    从字符串中查找子字符串的方法：in的Of()和lastIndexOf()，然后返回子字符串的位置(如果没有找到该子字符串，则返回-1)。

这两个方法都可以接收可选的第二个参数，表示从字符串中的哪个位置开始搜索。也就是说，indexOf()会从该参数指定的位置向后搜索，忽略该位置之前的所有字符；而lastIndexOf()则会从指定的位置向前搜索，忽略该位置之后的所有字符。

    4.trim()方法：

    trim()方法，这个方法会创建一个字符串的副本，删除前置及后缀的所有空格，然后返回结果，但不改变原来的字符串；
var str = ' hello world ';
var trimStr = str.trim();
alert(str);  //  " hello world "
alert(trimStr); // "hello world";

第 11 章 DOM 扩展

  • 选择符API：

    querySelector()方法接收一个CSS选择符，返回与该模式匹配的第一个元素，如果没有找到匹配的元素，返回null。

querySelectorALL() 和querySelector()方法一样，都是接收一个CSS选择符，querySelectorALL()返回的是所有匹配的元素，返回的是一个伪数组；

matchesSelector() 参数也是CSS选择符，如果条用元素与改选择符匹配，返回True,否则false。
function matchesSelector(element,selector){
    if(element.matchesSelector){
        return element.msMathesSelector(selector);
    }else if(element.msMatchesSelector){
        return element.msMatchesSelector(selector);
    }else if(element.mazMatchesSelector){
        return element.mozMatchesSelector(selector);
    }else if (element.mozMatchesSelector){
        return element.webkitMatchesSelector(selector);
    }else{
        throw new Error('Not supported.');
    }
}

if(matchesSelector(document.body,"body.page1")){
    //执行操作
}

    • splice(起点,长度)的用法：功能强大的很，可删除、插入、替换；

      var arr=[1,2,3,4,5,6];
//删除：splice(起点, 长度)
arr.splice(2, 3);  //1,2,6
//插入：splice(起点, 长度, 元素...);
// arr.splice(2, 0, 'a', 'b', 'c');  // 1,2,a,b,c,4,5,6
// arr.splice(2, 2, 'a', 'b');//1,2,a,b,5,6

var arr=[12, 8, 99, 19, 112];

- **sort()的用法：将数组中的元素颠倒顺序并返回排序后的数组。(改变原来的数组)；；**

        arr.sort(function (n1, n2){
            if(n1<n2){
                return -1;
            }else if(n1>n2){
                return 1;
            }else{
                return 0;
            }
        });

- **pop()的用法：从数组后边删除；**

        var arr=[1,2,3];
        arr.pop();// 1,2

- **push()的用法：从数组后边添加；**

        var arr=[1,2,3];
        arr.push(4);//1,2,3,4

- **shift()的用法：从数组头部删除；**

        var arr=[1,2,3];
        arr.shift();//2,3；

- **unshift()的用法：从数组头部添加；**

        var arr=[1,2,3];
        arr.unshift(4);//4，1,2,3；

- **join()的用法：将数组中所有元素都转化未字符串并连接在一起，返回最后生成的字符串。；**

- **reverse()的用法：将数组中的元素颠倒顺序，返回逆序的数组。(改变原来的数组)；**

- **sort()的用法：将数组中的元素颠倒顺序并返回排序后的数组。(改变原来的数组)；**


        var a = [1,33,4,2,44];
        a.sort(function (a,b){
            return a-b;
        });
        console.log(a);//1,2,4,33,44;

- **concat()的用法：创建并返回一个新数组.(不改变原来的数组)；**

- **slice(开始位置,结束位置(不包括结束位置的元素))的用法：返回指定数组的一个片段或子数组(不会修改数组)；**

3. ECMAScirpt5 中的数组方法

  • forEach()的方法：从头至尾遍历数组，为每个元素调用指定的函数。forEach()使用三个参数调用该函数：数组元素、元素的索引和数组本身。

    var data = [1,2,3,4,5];  //要 求和的数组
var sum = 0;
data.forEach(function (value){sum += value;});
sum  // 15;
data.forEach(fucntion (v,i,a){a[i] = v + 1; });
data  //[2,3,4,5,6]

  • map()的方法：将调用的数组的每个元素传递给指定的函数，并返回一个数组，它包含该函数的返回值。

    a = [1,2,3];
b = a.map(function (x) {return x*x;});//  b是[1,4,9];

  • filter()的方法：将调用的数组的每个元素传递给指定的函数，并返回一个数组，它包含该函数的返回值。

    a = [5,4,3,2,1];
smallvalues = a.filter(function(x){return x<3});  // [2,1]
everyother = a.filter(function(x,i){return i%2==0});  // [5,3,1];
其中，x为数组元素，i为数组索引，这里是可以有第三个参数的，第三个参数为元素本身；第二个和第三个参数可选。

另，filter()会跳过稀疏数组中缺少的元素，它的返回数组总是稠密的。为压缩稀疏数组的空缺：
a = a.filter(function(x){return x != undefined && x != null;})

  • every()和some()的方法：是数组的逻辑判断：它们对数组元素应用指定的函数进行判定，返回true或false.

    a = [1,2,3,4,5];
a.every(function(x){return x<10}); // true 
a.every(function(x){return x%2 == 0}); // false 

a.some(function(x){return x%2 == 0}); // true

在空数组上调用时，every()返回true,some()返回false.

  • reduce(函数，传递给函数的初始值)和reduceRight()方法用指定的函数将数组元素进行组合，生成单个值。

    var a = [1,2,3,4,5];
var sum = a.reduce(function (x,y){return x+y; 0});//15
var product = a.reduce(function (x,y){return x*y; 1});//120
var max = a.reduce(function (x,y){return x>y?x:y; });//5
    • reduceRight()和reduce()不同的是它是按照数组索引从高到低(从左到右)处理数组，而不是从低到高。

  • indexOf()和lastIndexOf()搜索整个数组中具有给定值的元素，返回找到的第一个元素的索引或者如果没有找到就返回-1。indexOf()从头至尾搜索，而lastIndexOf()则反向搜索。

    a = [0,1,2,1,0];
a.indexOf(1);   // 1
a.lastIndexOf(1);  //3
a.indexOf(3);  //-1

// 在数组中查找所有出现的x，并返回一个包含匹配索引的额数组
function findall(a,x){
       var result = [],
    len = a.length,
    pos = 0;   //开始搜索的位置
    while(pos<len){
        pos = a.indexOf(x,pos);
        if(pos === -1){
               break;
        }
           result.push(pos);
        pos += 1;
      }
   return result;
}
var arr = [4,3,23,4,]
console.log(findall(arr,4));// [0, 3]
